## 欢迎使用 Quest 1.0

Quest 1.0 是全球首个可实时自我学习与进化的自主编程智能体，让开发者从开发执行者变成任务管理者。

### 核心特点

- **Quest on, Hands off** — 委派任务后放手，AI 自主完成
- **自主执行** — AI 独立完成从需求澄清到代码交付的全流程
- **自我进化** — 从项目历史中学习，遇错自修，越用越懂
- **可验证交付** — 通过 SPEC 文档和测试保证质量

## 统一入口设计

Quest 1.0 采用**统一入口**，直接输入目标即可使用。Quest 会智能判断任务类型并选择最佳执行策略。

> **无需选择模式**：直接在输入框描述你的目标，Quest 会自动处理一切。

### 可选的引导场景

如需快速进入特定场景，可使用以下引导入口：

#### Explore an Idea

适合场景：
- Vibe Coding 创作
- 0-1 项目快速原型
- 一句话生成应用

特点：
- 不问技术细节，专注功能玩法
- 默认技术栈自动选择
- 实时预览，快速迭代
- 前端任务有 Good Taste

#### Code with Spec

适合场景：
- 存量工程维护和迭代
- 专业软件开发
- 复杂工程任务

特点：
- 智能判断是否需要生成 SPEC
- 支持多种执行环境（Local/Worktree/Cloud）
- 可验证的交付标准
- 多子 Agent 并行探索方案

## 核心能力

### 1. 任务编排

像管理团队一样管理 AI：
- 统一视图管理任务全生命周期
- 多任务并行执行
- 实时监控进度（To-do List、Changed Files）

### 2. 异步并行

打破时间空间限制：
- 云端任务可关机离线执行
- 多个任务独立沙箱同时运行
- 三种执行环境灵活选择

### 3. SPEC 驱动

用文档保证质量：
- AI 智能判断任务复杂度
- 复杂任务自动生成 SPEC
- 严格执行与验证

### 4. 项目洞察

深度理解代码库：
- 分析代码库结构和架构演进
- 积累项目知识库
- 基于历史上下文决策
