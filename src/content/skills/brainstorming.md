## Use Cases

- Product feature brainstorming
- Solution exploration
- Creative project kickoff
- Problem reframing
- Team creative collaboration

## Core Capabilities

- **Divergent Thinking**: Generate lots of ideas
- **Association Extension**: Expand from one idea
- **Reverse Thinking**: Explore possibilities in reverse
- **Cross-domain Inspiration**: Borrow from other fields

## Example

```
I need to brainstorm new features for Qoder:

Context:
- Target users: Independent developers
- Existing features: Code generation, completion, refactoring
- Competitive edge: Need unique selling points

Please use "Yes, and..." method to expand ideas,
generate 15 potential new feature concepts.
```

## Notes

- Quantity first, quality second
- Defer judgment, fully diverge
- Encourage wild ideas
- Record all ideas
