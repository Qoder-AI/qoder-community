## Use Cases

- Apply brand themes to presentations
- Unify document visual style
- Quick theme switching
- Generate custom themes
- Maintain brand consistency

## Core Capabilities

- **Preset Themes**: 10+ professional themes
- **Custom Themes**: Generate based on needs
- **One-click Apply**: Quick style changes
- **Multi-format Support**: Slides, docs, web pages

## Example

```
Please apply corporate blue theme to this report:
- Primary color: Deep blue
- Accent color: Gold
- Font: Professional business style
- Keep original content structure

Or, generate a tech theme for product launch.
```

## Notes

- Choose themes matching content
- Maintain consistent theme usage
- Can fine-tune based on presets
- Consider audience and occasion
