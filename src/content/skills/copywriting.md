## Use Cases

- Write ad copy
- Create landing page content
- Compose marketing emails
- Social media posts
- Product descriptions

## Core Capabilities

- **AIDA Framework**: Attention-Interest-Desire-Action
- **Emotional Triggers**: Leverage psychological principles
- **Value Proposition**: Clearly communicate product value
- **CTA Optimization**: Effective calls to action

## Example

```
Please write landing page copy for Qoder:

Target audience: Independent developers
Core value: AI-powered coding assistant
Differentiator: Smarter than Cursor
Goal: Drive free trial signups

Include: Headline, subheadline, 3 value points, CTA
```

## Notes

- Understand target audience pain points
- Keep it concise and powerful
- Test different versions
- Follow brand voice
