## Use Cases

- Create project implementation plans
- Write technical solution documents
- Plan product roadmaps
- Draft strategic plans
- Define milestones and deliverables

## Core Capabilities

- **Goal Definition**: Clear goal statements
- **Task Breakdown**: Break goals into actionable steps
- **Timeline Planning**: Reasonable time arrangements
- **Risk Identification**: Anticipate potential issues

## Example

```
Please help create a user auth system refactoring plan:

Goal: Migrate old session auth to JWT
Constraint: Cannot affect existing users
Timeline: 4 weeks

Include: Goals, phased tasks, risks and mitigations.
```

## Notes

- Plans should be executable, not too abstract
- Consider dependencies
- Leave buffer time
- Review and adjust regularly
