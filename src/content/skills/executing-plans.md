## Use Cases

- Execute projects according to plan
- Track task completion status
- Handle execution obstacles
- Adjust execution strategies
- Generate progress reports

## Core Capabilities

- **Task Execution**: Complete tasks per plan
- **Progress Tracking**: Monitor completion status
- **Problem Solving**: Handle execution barriers
- **Plan Adjustment**: Modify plans based on situation

## Example

```
I have the following plan to execute:
1. Design database schema (completed)
2. Implement API endpoints (in progress)
3. Write test cases (pending)
4. Deploy to test environment (pending)

Please help:
- Continue executing step 2
- Identify potential blockers
- Suggest next actions
```

## Notes

- Keep plan and execution in sync
- Communicate progress and issues promptly
- Be flexible with changes
- Do retrospective after completion
