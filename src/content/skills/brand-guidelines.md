## Use Cases

- Create brand-compliant designs
- Ensure marketing material consistency
- Apply correct colors and fonts
- Maintain brand visual identity
- Train team on brand usage

## Core Capabilities

- **Color System**: Official color specifications
- **Typography Standards**: Recommended fonts and usage
- **Spacing Rules**: Standard spacing system
- **Usage Examples**: Correct and incorrect usage comparison

## Example

```
Please check if this marketing email follows brand guidelines:
1. Are colors used correctly
2. Do fonts meet requirements
3. Is logo usage proper
4. Is overall style consistent
```

## Notes

- Always reference latest brand guide
- Don't arbitrarily modify brand colors
- Maintain sufficient space around logo
- Consult full guidelines if uncertain
