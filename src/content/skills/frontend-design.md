## Use Cases

- Design unique web interfaces
- Create React/Vue components
- Develop landing pages and marketing pages
- Beautify existing UIs
- Build design system components

## Core Capabilities

- **Design Thinking**: Understand context and goals
- **Unique Aesthetics**: Avoid cookie-cutter AI styles
- **Motion Design**: Micro-interactions and animations
- **Responsive**: Multi-device adaptation

## Example

```
Please help me design a pricing page component:
- 3 price tiers
- Style: Modern, minimal but distinctive
- Needs visual hierarchy
- Highlight recommended plan
- Use React + Tailwind CSS
```

## Notes

- Clarify design direction and style tone
- Don't use overused fonts like Inter
- Avoid purple gradients and other common AI colors
- Focus on details and subtle design choices
