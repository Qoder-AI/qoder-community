## Use Cases

- New user welcome sequences
- Trial to paid conversion
- Product feature introduction
- Churned user win-back
- Event promotion

## Core Capabilities

- **Sequence Design**: Appropriate email cadence
- **Content Strategy**: Goals for each email
- **Personalization**: Customize based on user behavior
- **A/B Testing**: Optimize subject lines and content

## Example

```
Please design a 7-day trial conversion email sequence:

Product: Qoder Pro
Trial period: 7 days
Goal: Convert to paid users

Include for each email:
- Send timing
- Subject line
- Core content
- CTA
```

## Notes

- Don't send too frequently
- Provide real value
- Clear unsubscribe option
- Comply with email marketing regulations
