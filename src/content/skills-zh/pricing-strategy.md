## 使用场景

- SaaS 定价设计
- 价格层级规划
- 竞品价格分析
- 促销策略
- 价格敏感度测试

## 核心能力

- **价值定价**：基于客户价值定价
- **层级设计**：合理的功能分层
- **竞争分析**：市场定位
- **心理定价**：价格呈现技巧

## 示例

```
请为 Qoder 设计定价策略：

产品：AI 编码助手
目标用户：独立开发者、创业公司、企业
竞品定价：
- Cursor：$20/月
- GitHub Copilot：$19/月

设计：
- 3-4 个价格层级
- 每层功能分配
- 定价建议和理由
- 年付折扣策略
```

## 注意事项

- 定价影响品牌定位
- 留出未来调价空间
- 考虑国际市场差异
- 保持简单透明
