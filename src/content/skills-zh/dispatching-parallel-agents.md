## 使用场景

- 并行处理多个独立任务
- 协调多 Agent 协作
- 加速大规模数据处理
- 实现复杂工作流
- 任务分解和结果合并

## 核心能力

- **任务分解**：将大任务拆分为子任务
- **并行执行**：同时运行多个 Agent
- **结果聚合**：合并各 Agent 输出
- **错误处理**：处理单个 Agent 失败

## 示例

```
我需要分析 10 个竞品网站，请帮我：
1. 分配任务给并行 Agent
2. 每个 Agent 分析一个网站
3. 收集结果并生成对比报告

分析维度：定价、功能、UI/UX、技术栈
```

## 注意事项

- 确保任务可以并行化
- 考虑资源限制
- 设计好结果合并策略
- 处理部分失败情况
