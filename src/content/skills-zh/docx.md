## 使用场景

- 自动化报告生成和格式设置
- 合同文档编辑和批注
- 批量文档处理和转换
- 从 Word 文档中提取结构化数据
- 协作文档的修订跟踪

## 核心能力

- **创建文档**：从零开始构建格式精美的 Word 文档
- **编辑内容**：修改现有文档的文本、样式和布局
- **修订跟踪**：启用和管理修订跟踪
- **添加批注**：在文档中插入注释和批注
- **提取文本**：从复杂文档中提取纯文本或结构化内容

## 示例

```
请帮我创建一份项目进度报告，包含：
1. 项目概述
2. 本周完成的任务（表格形式）
3. 下周计划
4. 风险和问题

使用专业的商务文档格式。
```

## 注意事项

- 支持 .docx 格式（Office 2007+）
- 复杂表格和图表可能需要额外处理
- 处理敏感文档时注意数据安全
