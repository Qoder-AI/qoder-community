## 使用场景

- 个人任务管理
- GTD 工作流
- 项目规划
- 日程管理
- 回顾和复盘

## 核心能力

- **任务创建**：快速添加待办事项
- **项目管理**：将任务组织到项目中
- **标签系统**：灵活的分类方式
- **截止日期**：时间管理

## 示例

```
请规划我下周的工作任务：

项目：Qoder 发布
截止日期：周五

任务清单：
1. 完成 UI 调整（周二）
2. 测试所有功能（周三）
3. 准备发布公告（周四）
4. 正式上线（周五）

请在 Things 3 中创建这些任务。
```

## 注意事项

- 仅支持 macOS
- 需要安装 Things 3
- 使用 URL Scheme 集成
- 保持任务描述简洁
