## 使用场景

- AI Agent 性能评估
- 模型输出质量测试
- A/B 测试框架
- 回归测试
- 基准测试

## 核心能力

- **指标定义**：定义评估指标
- **测试用例**：设计测试场景
- **结果分析**：解读评估结果
- **持续监控**：建立监控机制

## 示例

```
请为代码生成 Agent 设计评估框架：

评估维度：
1. 代码正确性 - 能否通过测试
2. 代码质量 - 可读性、最佳实践
3. 响应时间 - 生成速度
4. 一致性 - 相同输入的稳定性

提供：
- 具体评估指标
- 测试用例示例
- 评分标准
- 自动化测试方案
```

## 注意事项

- 选择有意义的指标
- 测试用例要有代表性
- 定期更新评估基准
- 避免过拟合评估集
