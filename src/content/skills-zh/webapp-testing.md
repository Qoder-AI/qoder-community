## 使用场景

- 端到端功能测试
- UI 回归测试
- 跨浏览器兼容性测试
- 视觉测试和截图对比
- 用户流程自动化验证

## 核心能力

- **浏览器自动化**：模拟用户操作
- **截图捕获**：保存页面状态
- **日志查看**：获取浏览器控制台日志
- **多浏览器支持**：Chrome、Firefox、Safari

## 示例

```
请帮我编写登录流程的 Playwright 测试：
1. 导航到登录页面
2. 输入用户名和密码
3. 点击登录按钮
4. 验证成功跳转到首页
5. 验证用户名正确显示
```

## 注意事项

- 使用稳定的选择器（data-testid）
- 添加适当的等待机制
- 隔离测试数据
- 清理测试状态
