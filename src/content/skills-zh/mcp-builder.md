## 使用场景

- 构建自定义 MCP 服务器
- 集成第三方 API 和服务
- 创建数据库访问工具
- 实现文件系统操作
- 扩展 AI Agent 能力

## 核心能力

- **工具设计**：定义清晰有效的工具接口
- **Python 实现**：使用 FastMCP 框架
- **TypeScript 实现**：使用 MCP SDK
- **最佳实践**：错误处理、安全性、性能

## 示例

```
请帮我创建一个 MCP 服务器，包含以下工具：
1. 查询天气 API
2. 搜索新闻
3. 翻译文本

使用 Python FastMCP 实现，包含完整的错误处理。
```

## 注意事项

- 遵循 MCP 协议规范
- 妥善保管 API 密钥
- 实现适当的速率限制
- 提供清晰的工具描述
