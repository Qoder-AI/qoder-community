## 使用场景

- Issue 创建和管理
- 项目进度跟踪
- Sprint 规划
- 工作流自动化
- 团队协作

## 核心能力

- **Issue 操作**：创建、更新、关闭
- **项目管理**：查看项目状态
- **标签分类**：自动标签
- **状态流转**：自动化状态变更

## 示例

```
请在 Linear 中创建以下任务：

项目：Qoder 2.0
Sprint：当前 Sprint

任务：
1. 实现用户认证模块 [高优先级, 后端]
2. 设计新的落地页 [中优先级, 设计]
3. 编写 API 文档 [低优先级, 文档]

每个任务包含简要描述。
```

## 注意事项

- 需要配置 Linear API key
- 确保项目权限
- 遵循团队工作流规范
- 保持 Issue 描述清晰
