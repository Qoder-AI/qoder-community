## 使用场景

- 动态节点数据引用
- 数据字段映射
- 条件逻辑表达式
- 日期时间处理
- 字符串操作

## 核心能力

- **数据引用**：$json、$node 语法
- **函数调用**：内置函数使用
- **条件表达式**：三元运算和逻辑运算符
- **链式访问**：安全的属性访问

## 示例

```
常用表达式示例：

// 访问当前节点数据
{{ $json.fieldName }}

// 访问其他节点数据
{{ $node["NodeName"].json.field }}

// 条件表达式
{{ $json.status === "active" ? "是" : "否" }}

// 日期格式化
{{ $now.format("YYYY-MM-DD") }}

// 安全访问嵌套字段
{{ $json.user?.address?.city ?? "未知" }}
```

## 注意事项

- 表达式区分大小写
- 正确处理空值
- 使用 ?? 设置默认值
- 测试边界情况
