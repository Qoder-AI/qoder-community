## 使用场景

- 设计独特的 Web 界面
- 创建 React/Vue 组件
- 开发落地页和营销页面
- 美化现有 UI
- 构建设计系统组件

## 核心能力

- **设计思维**：理解上下文和目标
- **独特美学**：避免千篇一律的 AI 风格
- **动效设计**：微交互和动画
- **响应式**：多设备适配

## 示例

```
请帮我设计一个定价页面组件：
- 3个价格层级
- 风格：现代、简约但有特色
- 需要视觉层次
- 突出推荐方案
- 使用 React + Tailwind CSS
```

## 注意事项

- 明确设计方向和风格调性
- 不要使用过度使用的字体如 Inter
- 避免紫色渐变等常见 AI 配色
- 注重细节和微妙的设计选择
