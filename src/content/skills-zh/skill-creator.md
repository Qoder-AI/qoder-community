## 使用场景

- 创建自定义 Skills
- 封装专业知识
- 分享可复用能力
- 扩展 AI Agent 功能
- 团队知识传递

## 核心能力

- **结构指南**：SKILL.md 文件规范
- **最佳实践**：有效的 Skill 设计
- **渐进披露**：Token 高效的架构
- **发布流程**：分享和分发

## 示例

```
请帮我创建一个新的 Skill：

名称：react-testing
描述：React 组件测试最佳实践
目的：指导编写高质量的 React 测试

生成完整的 SKILL.md 文件，包括：
- YAML frontmatter
- 使用场景
- 核心指令
- 示例用法
```

## 注意事项

- SKILL.md 保持简洁（< 5000 tokens）
- 描述清晰便于触发
- 测试 Skill 效果
- 参考官方示例
