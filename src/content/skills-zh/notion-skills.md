## 使用场景

- 知识库管理
- 项目跟踪
- 会议记录
- 文档协作
- 数据库操作

## 核心能力

- **页面操作**：创建、更新、查询
- **数据库管理**：CRUD 操作
- **内容格式**：丰富的 Block 类型
- **关联链接**：页面间关联

## 示例

```
请在 Notion 中创建一个项目管理模板：

包含：
1. 项目概览页
2. 任务数据库（状态、负责人、截止日期）
3. 文档库
4. 会议记录模板

数据库之间需要建立关联。
```

## 注意事项

- 需要配置 Notion API key
- 注意 API 速率限制
- 准确设置权限
- 大量操作需要批量处理
