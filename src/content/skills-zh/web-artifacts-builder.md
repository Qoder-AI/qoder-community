## 使用场景

- 构建复杂的 React 组件
- 创建多页面应用
- 使用 shadcn/ui 组件库
- 状态管理和路由
- 交互式原型

## 核心能力

- **组件架构**：多文件组件结构
- **状态管理**：React state 和 context
- **UI 库集成**：shadcn/ui 组件
- **样式系统**：Tailwind CSS

## 示例

```
请构建一个项目管理仪表板：

功能：
1. 项目列表视图
2. 任务看板（拖拽）
3. 团队成员管理
4. 数据图表

技术栈：React + Tailwind + shadcn/ui
要求：响应式设计，支持暗色模式
```

## 注意事项

- 适用于复杂的多组件应用
- 简单单文件使用 frontend-design
- 注重组件复用性
- 保持代码结构清晰
