## 使用场景

- 创建 Slack 表情 GIF
- 制作团队庆祝动画
- 设计产品演示 GIF
- 创作有趣的反应图
- 社交媒体内容制作

## 核心能力

- **尺寸优化**：满足 Slack 尺寸限制
- **动画设计**：流畅的动画效果
- **循环处理**：完美循环动画
- **文件压缩**：控制文件大小

## 示例

```
请创建一个庆祝的 Slack GIF：
- 内容：五彩纸屑效果
- 文字："SHIPPED!"
- 尺寸：适合 Slack 表情
- 循环播放
- 时长：2秒
```

## 注意事项

- Slack 自定义表情最大 128KB
- 推荐尺寸 128x128 像素
- 限制帧数以减小文件大小
- 测试在 Slack 中的实际显示效果
