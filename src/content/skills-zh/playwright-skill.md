## 使用场景

- 端到端自动化测试
- 网页爬取和数据抓取
- 自动化表单填写
- 页面截图和 PDF 生成
- 性能测试

## 核心能力

- **多浏览器支持**：Chromium、Firefox、WebKit
- **页面交互**：点击、输入、等待
- **网络拦截**：模拟 API 响应
- **设备仿真**：模拟移动设备视口

## 示例

```
请编写一个 Playwright 脚本：
1. 打开电商网站
2. 搜索特定商品
3. 添加到购物车
4. 截图保存结果
5. 导出商品信息
```

## 注意事项

- 使用显式等待而非固定延迟
- 处理弹窗和 iframe
- 注意反爬虫措施
- 设置合理的超时时间
