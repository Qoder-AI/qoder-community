## 使用场景

- AWS 资源创建
- CDK 基础设施代码
- Lambda 函数开发
- S3、DynamoDB 操作
- 安全最佳实践

## 核心能力

- **CDK 开发**：TypeScript/Python 基础设施
- **Serverless**：Lambda、API Gateway
- **存储服务**：S3、DynamoDB、RDS
- **安全配置**：IAM、VPC、安全组

## 示例

```
请创建一个 Serverless API：

需求：
- API Gateway + Lambda
- DynamoDB 作为数据存储
- S3 用于文件上传
- CloudWatch 日志和告警

使用 CDK TypeScript 编写。
```

## 注意事项

- 遵循最小权限原则
- 使用环境变量管理配置
- 注意成本管理
- 配置适当的监控
